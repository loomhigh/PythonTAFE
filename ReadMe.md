## Session 1

### Introduction to Programming Workshop
• Introduction to programming
• Sequence, selection & iteration overview
• Pseudocode overview
• Variables
• Data types
• String concatenation & interpolation

### Out of Class Activity

It is expected that learners should complete the following in
their own time after the session. Review the material
covered during this session to ensure you understand the
concepts and principles covered
• finish any work not completed during class time.
• complete any pre-reading for the next session
What is required will be discussed with students at the end
of the session. This should take around an hour
